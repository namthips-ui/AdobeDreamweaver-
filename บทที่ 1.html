ความรู้เบื้องต้นการสร้างเว็บไซต์
 

Web Page คือ เอกสารข้อมูลที่แสดงผลในรูปแบบของภาษา HTML (HyperText Markup Language) ซึ่งสามารถเปิดดูได้ผ่านเว็บเบราว์เซอร์ โดยเว็บเพจแต่ละหน้ามักมีการเชื่อมโยงกับหน้าอื่นๆ ผ่านทางไฮเปอร์ลิงก์ (Hyperlink) ทำให้ผู้ใช้สามารถเข้าถึงข้อมูลที่เกี่ยวข้องได้อย่างสะดวกรวดเร็ว

Website คือ กลุ่มของเว็บเพจที่ถูกรวบรวมไว้อยู่ภายใต้โดเมนเนม (Domain Name) เดียวกัน โดยแสดงเนื้อหาในรูปแบบที่เกี่ยวข้องหรือเป็นระบบเดียวกัน ตัวอย่างของโดเมนเนม เช่น www.hellomyweb.com ทั้งนี้ คำว่า “เว็บไซต์” บางครั้งอาจใช้เรียกแทนชื่อโดเมนเนมโดยรวมได้เช่นกัน

Home Page หรือ “หน้าแรกของเว็บไซต์” คือหน้าเว็บเพจหลักที่จะแสดงขึ้นมาเป็นอันดับแรกเมื่อผู้ใช้เข้าสู่เว็บไซต์ เช่น เมื่อผู้ใช้พิมพ์ที่อยู่ www.hellomyweb.com ระบบจะแสดงหน้าเว็บเพจที่มีชื่อไฟล์ว่า index เป็นหน้าแรกโดยอัตโนมัติ หน้าแรกนี้มักทำหน้าที่เป็นสารบัญหรือจุดเริ่มต้นในการเข้าถึงเนื้อหาต่างๆ ภายในเว็บไซต์
